1.git是什么？
git是分布式版本控制器。
2.常用git命令？
git init 将该目录变成git可以管理的仓库。
git add start.md 将文件start添加到仓库
git commit -m"XXX" 把文件提交到仓库
git remote add origin ssh  关联一个远程库
git push -u origin master 将本地内容推送到远程（第一次推送master分支的所有内容，之后git push origin master推送最新修改；）
git status
